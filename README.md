
# ContactList

A simple CRUD application built with the MERN stack.

- CRUD: Create, Read, Update, Delete.
- MERN: MongoDB, Express.js, React, Node.js.
## Installation

Download [Node.js](https://nodejs.org/en/download). Run the following commands :

for the backend :

```bash
  cd backend
  npm install
  npm start
```

for the frontend :

```bash
  cd frontend
  npm install
  npm run dev
```
